PRODUCT INTEGRATION TESTING (V3DSTS)
This new testing facility, called “Production Integration Testing”, also referred to as the “V3DSTS”, is a remote web-based testing environment that provides the equivalent of all components of the 3-D Secure infrastructure. The V3DSTS provides Members (and Member processors), merchants (and merchant aggregators), and vendors with a 3-D Secure environment for production readiness testing of ACS and MPI component implementations. All Visa regions have mandated that new 3-D Secure implementations successfully complete V3DSTS testing prior to entering production.

HOURS OF OPERATION:
The automated Production Integration Testing facility is scheduled to be available for ACS and MPI component implementation testing on a 24X7 basis, 365 days a year except for scheduled and emergency maintenance.
TESTING APPROACH
The Production Integration Testing (V3DSTS) environment utilizes a pre-defined,published set of test cases representing numerous transaction processing scenarios essential to the quality operation of the 3-D Secure service.

These test cases are designed to ensure that new component implementations utilize proper message formats, correctly populate data and accurately process the 3-D Secure transaction test messages.

In addition, some of the other design features of the V3DSTS are:
· Support for Version 1.0.2 of the 3-D Secure protocol as well as backward compatibility with Version 1.0.1. Earlier versions are not supported

· Access, by the Visa regional offices, to the results of V3DSTS testing conducted by user testing entities within each of their regions

· Support for “proof of attempts” testing

· Support for CAVV testing from an Issuer and Acquirer perspective (Contact your local Visa region for more information)

· Notification, to the respective Visa regional office, at the time of user enrollment in the V3DSTS and upon user indication that V3DSTS testing has been completed (at the option of the Visa region)
